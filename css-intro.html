What is CSS?

CSS is an acronym for Cascading Style Sheet. Used in conjunction with HTML. Customized the elements and layout of a webpage

Imagine building a website as building a house

  • HTML = structure (ex. floor, roof, walls)
  • CSS = decorations (ex. paint, furniture)
  • Javascript = function (ex. lighting, plumbing)

There are three way to include CSS for a webpage:

  • Inline: useful for adding a few custom element
  • Internal: we create a pair of a style elements in the head of our HTML document, useful for a single page website
  • External: we can link a CSS file to our HTML file to style one or all pages of a website, useful for many pages website

Rules for declaring CSS property

  • Identify the html element
  • The list of properties are separated by a semicolon and enclosed by curly braces
  • Specify the ‘value’ for a ‘property’ using a colon
